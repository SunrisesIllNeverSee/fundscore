# Security Policy

## Reporting a vulnerability

If you discover a security vulnerability, please **do not** open a public issue.

Report privately via [GitHub Security Advisories](https://github.com/SunrisesIllNeverSee/fundscore/security/advisories/new).

## Scope

- The fundscore CLI
- The MCP server (stdio transport)
- The scoring engine and check logic
- The GitHub Action workflow

## Out of scope

- The signalaf.com web app (report at [sigrank-app](https://github.com/SunrisesIllNeverSee/sigrank-app/security))
- Third-party dependencies (report upstream)

## Response timeline

- Acknowledgement within 48 hours
- Assessment within 7 days
